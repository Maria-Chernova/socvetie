🌿 Учебный проект

Сайт «Соцветие» — это демонстрационный проект, созданный для портфолио веб-разработчика.

  • Магазин не существует и не оказывает услуг
  • Форма обратной связи не сохраняет и не обрабатывает данные
  • Отправленные заявки никуда не передаются
  • Все фото взяты с Unsplash по свободной лицензии
  • Все контактные данные вымышлены

Цель проекта — показать навыки создания адаптивных веб-сайтов на Flask + Bootstrap.

🔒 Демонстрационная политика обработки данных

Важно: этот сайт является учебным проектом для портфолио.

Как обрабатываются данные из формы:
  • Данные не сохраняются ни в базу данных, ни в файлы
  • Данные не выводятся в консоль сервера
  • Данные не передаются третьим лицам
  • После нажатия кнопки «Отправить» данные полностью удаляются
  • Форма существует исключительно для демонстрации интерфейса
В реальном проекте:

При создании коммерческого сайта необходимо:

  • Разработать и опубликовать Политику обработки персональных данных
  • Получить явное согласие пользователя (чекбокс)
  • Обеспечить защиту данных согласно 152-ФЗ
  • Хранить данные на сервере на территории РФ
  • Предоставить возможность удаления данных по запросу
